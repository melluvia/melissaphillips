DishRate App for iOS

Dish Rate is the fun way to share favorite dishes with others. Users can instantly upload and rate any dish at any time with a familiar graphic star rating system. The loading screen depicting the sharing of a great meal subtly becomes a blurred background promoting the idea that the app supports a community of users, allowing others to browse through saved dishes and try and rate them as well. Each dish created accommodates customizations for the title, rating, an optional note, and the ability to upload a photo from the device's camera or stored photos and save the dish for future reference and further editing by the initial creator if desired. Users can get a fresh perspective by browsing through dishes added by the community and use the average rating feature to help them decide what to try. The app allows users to create an account for cloud-based data storage, or skip account creation in order to use local storage on their device for personal use. It also allows the user to choose an easy login with facebook credentials. Dish Rate helps users discover new things to try whether cooking at home or dining out.

Color Me a Mentor App for iOS

This application won second place at the Women In Tech Demo Day competition in Plano, TX. It was designed specifically for Girls inc. which is a non-profit organization that is "inspiring all girls to be strong, smart and bold." The app allows Girls inc. members and prospective volunteer mentors to take a fun personality quiz which generates an individual color combination based on their personality type. This quiz will allow prospective mentors to get an instant sense of connection so that they will see themselves in Girls inc. members and feel called to mentor them. Designed for girls under the age of 20, a highly customizable interface with bright colors and fun avatars make this app the favorite of tween girls.

  • Date: October 2016
  • Service: iOS App Design & Development, Visual Design, Logo design

Local Weather App for Web

This responsive web app uses JQuery and the OpenWeather API to access the user's location and provide them with the local temperature with the option to toggle the temperature unit between Farenheit and Celcius. The app automatically displays a representational full screen background image that changes when temperature levels rise or fall significantly so that it always corresponds to the current weather condition.

Quotes On Design for Web

Quotes On Design is an interactive web app which allows the user to generate a variety of inspiring and amusing quotes by thinkers and creatives with the added option to tweet the quote. The quotes are accessed using the Quotes On Design API curated by Chris Coyier. The image is paired with a seamless background color so that at any screen size it appears as if the text is written on the wall of a creative workspace.
